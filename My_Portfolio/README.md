# Abhishek Portfolio (React + Vite)

A personal portfolio website built with React and Vite to showcase skills, certifications, and real projects with GitHub and live links.

## Highlights

- Clean single-page portfolio layout
- Animated sections and interactive UI
- Projects section with separate GitHub and Live Demo buttons
- Certificates gallery with modal preview
- Contact and social links
- Responsive design for desktop and mobile

## Tech Stack

- React 18
- Vite 5
- JavaScript (ES Modules)
- CSS3
- Font Awesome

## Project Structure

- `src/` - React app source code
- `src/components/` - UI sections/components
- `src/data/portfolioData.js` - Content for skills, certificates, projects, and links
- `public/` - Static assets (images, resume, badges)

## Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Run development server

```bash
npm run dev
```

### 3) Build for production

```bash
npm run build
```

### 4) Preview production build

```bash
npm run preview
```

## Customize Content

Edit `src/data/portfolioData.js` to update:

- skills
- services
- certificates
- projects (title, image, githubLink, liveLink, description)
- social links
